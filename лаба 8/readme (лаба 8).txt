============================
           EXPLANATION
============================

1. Account
----------
Приватные поля: __number, __balance, __pin
Методы:
- deposit(amount, pin) - пополнение счёта после проверки PIN
- withdraw(amount, pin) - снятие средств после проверки PIN
- get_balance(pin) - получение баланса после проверки PIN
Пример: 
acc = Account("12345", 1000, 1111)
acc.deposit(500, 1111)  # увеличит баланс до 1500

2. Product
----------
Приватная цена: __price
Методы:
- set_discount(percent) - устанавливает скидку
- final_price() - возвращает цену после скидки
Скидка не может быть отрицательной и не превышает 100%.

3. Course
---------
Приватные поля: название, список студентов, макс. кол-во мест
Методы:
- add_student(name) - добавить студента
- remove_student(name) - удалить студента
- get_students() - вернуть список студентов (копия)
Проверяется, чтобы курс не был переполнен и студент не добавлялся дважды.

4. SmartWatch
-------------
Приватное поле: __battery
Методы:
- use(minutes) - уменьшает заряд 1% за каждые 10 минут
- charge(percent) - добавляет заряд, max 100%
- get_battery() - возвращает текущий заряд

5. Transport (Bus, Train, Airplane)
-----------------------------------
Метод travel_time(distance) вычисляет время поездки
Airplane уменьшает время на 20% для перелётов

6. Order (DineInOrder, TakeAwayOrder, DeliveryOrder)
----------------------------------------------------
Метод calculate_total() учитывает разные надбавки:
- DineInOrder +10%
- TakeAwayOrder +5%
- DeliveryOrder +15%

7. Character (Warrior, Mage, Archer)
------------------------------------
Метод attack() реализован по-разному для каждого класса
Пример:
Warrior("Артур", 100, 25).attack() -> "Артур атакует мечом с силой 25!"

8. MediaFile (AudioFile, VideoFile, Podcast)
--------------------------------------------
Метод play() выводит разный текст в зависимости от типа файла

9. PaymentSystem (CreditCardPayment, CryptoPayment, BankTransfer)
-----------------------------------------------------------------
Абстрактный класс process_payment(amount)
Разная логика для карт, крипто и банковских переводов

10. Animal (Lion, Elephant, Snake)
---------------------------------
Методы eat() и sleep() реализованы по-разному
Пример:
Lion().eat() -> "Лев ест мясо, охотясь на добычу."

11. Document (WordDocument, PdfDocument, SpreadsheetDocument)
------------------------------------------------------------
Методы open(), edit(), save() реализованы для каждого типа документа

12. Lesson (VideoLesson, QuizLesson, TextLesson)
-------------------------------------------------
Метод start() выводит начало урока с разной информацией

13. Notification (EmailNotification, SMSNotification, PushNotification)
-------------------------------------------------------------------------
Метод send(message) выводит сообщение разными способами

14. Shape (Square, Circle, Triangle)
------------------------------------
Метод perimeter() считает периметр
Можно создавать список разных фигур и вызывать perimeter() для всех

15. Employee (Manager, Developer, Designer)
-------------------------------------------
Метод work() реализован для каждого типа сотрудника
Пример: Manager().work() -> "Менеджер планирует задачи и контролирует команду."
